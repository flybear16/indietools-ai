## 三、第一阶段：立即启动（0 成本）

### 3.1 Affiliate 分佣（最快变现）

**现状**：数据库已有 `affiliateUrl` 字段，3 个工具已标记 `affiliateEnabled: true`

**操作步骤**：
1. 逐个注册收录工具的 Affiliate / Referral 计划
2. 获取推广链接，填入 `affiliateUrl` 字段
3. 在工具详情页展示"访问"按钮时使用 affiliate 链接
4. 页面底部标注 "包含推广链接" 保持透明

**重点目标工具**（佣金高+用户多）：
- GitHub Copilot — $10/月订阅，可能有推荐计划
- Cursor — 有 referral 计划
- Vercel — 有 referral 计划（$10-20/注册）
- Supabase — 有 partner 计划
- Notion AI — 有 affiliate 计划（$5-10/注册）
- Jasper AI — 高佣金（$50-125/注册）
- Semrush — 高佣金（$10-200/注册）
- Ahrefs — 有 affiliate 计划（$10-50/注册）
- Canva — 有 affiliate 计划（$10-36/注册）
- Mailchimp — 有 referral 奖励

### 3.2 "Submit & Boost" 双轨制

- 免费提交 → 进入 pending 审核队列
- 付费 $29 → 加速审核 + 30天 Featured 展示
- 已有基础：submit 页面、pending 状态、isFeatured 字段
- 只需接入 Stripe 支付

### 3.3 Featured 展示位

- 工具方付 $49-$149/周 获得 "Featured" 标记 + 首页展示
- 在首页和 about 页加 "Sponsor / Advertise" 入口
- 简单的 Calendly 链接即可开始接单

## 四、第二阶段：流量增长后（1-3 个月）

### 4.1 Pro 会员（$9.9/月 或 $79/年）

**付费功能**：
- 🔍 高级筛选（技术栈、API 支持、团队规模、价格区间）
- 📊 工具对比（side-by-side comparison）
- ⭐ 收藏夹 + 个人工具箱
- 📧 每周精选工具邮件（含深度测评）
- 🎯 按项目阶段智能推荐工具组合

**原则**：免费版要好用，付费版是增值，不能让免费版变残废

### 4.2 Newsletter 广告位

- 建邮件列表 → 发 "AI Tools 周刊"
- 广告定价：
  - 头条推荐：$200-500/期
  - 工具推荐位：$50-150/期
- 前提：邮件订阅 > 2000

### 4.3 Sponsor 落地页

- 工具方每月 $99-$499 购买专属详情页增强版
- 包含：视频演示、用户案例、API 文档链接、专属优惠码
- 相当于工具方在 IndieTools 上开 "Mini 官网"
